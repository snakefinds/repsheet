Google Sheet Scraper - Portable EXE

What is included:
- GoogleSheetScraper.exe
- Run Google Sheet Scraper.bat

How to use:
1) Double-click "Run Google Sheet Scraper.bat" (or GoogleSheetScraper.exe).
2) Paste your Google Sheet link and API key.
3) Choose output JSON and click Scrape.

Notes:
- Python is NOT required on the target PC.
- This build is for Windows x64.
- If SmartScreen warns, click "More info" -> "Run anyway".
